# portfolio-model-2

Next.js portfolio with admin dashboard and Supabase integration.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Environment

Copy `.env.local.example` values into `.env.local` (not committed) with your Supabase credentials before running locally.

## Deploy

Deploy on [Vercel](https://vercel.com/new) or any platform that supports Next.js.
